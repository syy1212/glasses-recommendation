# 镜片品牌与系列推荐参考

> 目的：在完成验光数据闸门、镜框几何和折射率判断后，给出更稳妥的品牌/系列候选。  
> 更新基准：截至 `2026-07-05` 的公开官方产品信息。若用户问“最新款/刚发布/某地是否可以买到”，必须优先重新查品牌官方页或当地渠道。

## 1. 使用原则

- 品牌推荐必须排在数据可信度、镜框参数、折射率和镜片档位之后。
- 品牌只能作为“候选方向”，不要把单一品牌说成唯一正确答案。
- 同一品牌不同国家/地区的产品名、折射率、膜层和上市状态可能不同；不确定时说“当地渠道需确认”。
- `Yellow` 或 `Red-conservative-shopping` 场景只给普通单光/中端方向，不给高端定制定案。
- `Red-strict` 场景不推荐品牌，先建议线下专业验配或就医。
- 儿童近视管理、渐进/老花、抗疲劳/近用支持、变色/偏光，都不是成人普通单光默认项；只有用户场景命中时才展开。

## 2. 按档位给品牌候选

### 档位 1：普通单光实用型

适合：
- 预算优先
- `Yellow` 或保守建议
- 普通日常近视/近视散光
- 不追求高端定制和复杂定位数据

候选方向：
- 明月：`PMC`、常规非球面、`1.71` 性价比线
- 万新：常规单光、基础膜层/升级膜层线
- 凯米：`U2` 等常见性价比单光线
- 依视路：基础单光或当地门店常规单光线

推荐话术：
- “先把处方和镜框做稳，这一档已经够日常使用。”
- “不要为了品牌越级牺牲镜框尺寸、加工质量或售后。”

### 档位 2：中端升级型

适合：
- `Green` 或较强 `Yellow`
- 想要更好的膜层、通透感、耐污维护
- 预算中等，但还不具备高端定制定位条件

候选方向：
- 蔡司：标准单光、`ClearView`、当地可买的中端单光线
- 依视路：中端单光、`Crizal` 膜层体系相关产品
- 豪雅：常规单光/中端单光线，按当地产品名确认
- 明月：高性价比升级单光线

推荐话术：
- “这一档主要买膜层、稳定品控和售后，不是买玄学提升。”
- “如果镜框偏大，先缩框；不要只靠升级品牌补救。”

### 档位 3：高端定制/高端个性化

适合：
- `Green`
- 镜框稳定、小而居中
- 有瞳高或可靠定位数据
- 用户理解高端镜片收益依赖验配质量

候选方向：
- 蔡司：`ClearMind Single Vision`、`SmartLife`、当地高端定制单光线
- 罗敦司得：`B.I.G. EXACT / B.I.G. NORM` 等生物识别/个性化体系
- 豪雅：个性化定制类镜片，按当地产品名和验配能力确认
- 依视路：高端个性化/高端膜层组合，按当地产品名确认

谨慎点：
- 不能只因为预算够就推荐。
- 没有瞳高、镜框不稳、只拿电脑验光，先降级到中端或普通单光。
- 这类产品的关键不是“贵”，而是处方、定位、加工、售后一整套是否可靠。

## 3. 按特殊需求给品牌候选

### 变色片

适合：
- 经常室内外切换
- 不想频繁换太阳镜
- 接受车内变色效果受挡风玻璃影响、室内外恢复速度和底色差异

候选方向：
- `Transitions GEN S`
- 蔡司 `PhotoFusion X`
- 豪雅 `Sensity`

不默认推荐：
- 夜间驾驶为主
- 对室内底色非常敏感
- 只想解决电脑疲劳

### 数码/近用支持

适合：
- 近距离工作多
- 普通单光看近疲劳明显
- 年轻老视或调节压力明显，且经过验光确认

候选方向：
- 蔡司 `Digital ClearMind`
- 依视路 `Eyezen`
- 豪雅 `Sync III`

谨慎点：
- 不要把“每天看电脑”自动等同于必须上数码支持镜片。
- 先排查处方准确性、旧镜下滑、屏幕距离、干眼和休息习惯。

### 儿童近视管理

候选方向：
- 蔡司 `MyoCare`
- 豪雅 `MiYOSMART / MiYOSMART iQ`
- 依视路 `Stellest`

处理规则：
- 归入儿童近视防控/专项验配，不进入成人单光推荐流程。
- 必须建议线下专业验配和定期复查。
- 不要基于成人配镜逻辑直接推荐给儿童。

### 渐进/老花

候选方向：
- 依视路 `Varilux XR series`
- 蔡司渐进线，包括 `Progressive ClearMind`
- 罗敦司得个性化渐进体系
- 豪雅渐进线，按当地产品名确认

处理规则：
- 不属于普通成人单光近视推荐。
- 需要更完整的验配数据、试戴和售后适配。

## 4. 品牌推荐输出模板

```markdown
## 镜片品牌/系列建议
- 推荐档位：
- 为什么是这一档：
- 可选品牌/系列：
- 不建议越级的原因：
- 购买时要确认：
  - [ ] 是否有对应折射率
  - [ ] 是否适配当前镜框尺寸和片高
  - [ ] 膜层/变色/数码支持是否真是你的需求
  - [ ] 是否能提供可靠加工、售后和验货信息
```

## 5. 最新产品核验规则

用户问以下内容时，先查官方来源再回答：
- “最新款”
- “刚发布”
- “今年新款”
- “某系列是否还在卖”
- “某地区能不能买到”
- “某产品和另一个产品哪个更新”

优先来源：
- 品牌官方产品页
- 品牌官方新闻稿
- 当地官方渠道或经销商资料

不要仅依赖：
- 电商标题
- 社交平台转述
- 无法访问正文的攻略文章
